# Exercices : Le Second Degré

## Exercice 1 : Équations du second degré
Résoudre dans $\mathbb{R}$ les équations suivantes :
1. $2x^2 - 5x + 2 = 0$
2. $x^2 + 4x + 4 = 0$
3. $3x^2 + x + 2 = 0$

## Exercice 2 : Forme canonique
Déterminer la forme canonique des fonctions suivantes :
1. $f(x) = x^2 - 4x + 7$
2. $g(x) = 2x^2 + 8x - 10$

## Exercice 3 : Signe d'un trinôme
Étudier le signe des expressions suivantes selon les valeurs de $x$ :
1. $P(x) = x^2 - 3x - 4$
2. $Q(x) = -2x^2 + 4x - 2$

## Exercice 4 : Problème d'optimisation
Une entreprise produit $x$ objets par jour. Le coût de production est donné par $C(x) = 0,5x^2 - 40x + 1000$ (en euros).
1. Combien d'objets faut-il produire pour minimiser le coût ?
2. Quel est ce coût minimal ?
