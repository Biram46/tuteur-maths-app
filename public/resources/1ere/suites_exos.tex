\section{Suites Arithmétiques}
Définie par $u_{n+1} = u_n + r$.
Terme général : $u_n = u_0 + nr$.

\section{Suites Géométriques}
Définie par $u_{n+1} = u_n \times q$.
Terme général : $u_n = u_0 \times q^n$.

\section{Sommes}
Somme arithmétique : $S = \frac{(n+1)(u_0+u_n)}{2}$.
Somme géométrique : $S = u_0 \frac{1-q^{n+1}}{1-q}$.
