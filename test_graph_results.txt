Question: Peux-tu m'aider à dresser le tableau de signes de la fonction f(x) = (x-2)(x+3) ?

Envoi de la requête à Perplexity (Modèle: sonar)...


✅ RÉPONSE DU TUTEUR :

Je vais t'aider à dresser ce tableau de signes ! Avant de te donner la solution, posons-nous quelques questions pour bien comprendre la méthode.

**Questions guides :**
- Quelles sont les valeurs qui annulent chaque facteur $(x-2)$ et $(x+3)$ ?
- Comment varie le signe d'une fonction affine comme $(x-2)$ ?

---

## Méthode pas à pas

**Étape 1 : Trouver les racines**

Les valeurs qui annulent la fonction sont celles qui rendent chaque facteur égal à zéro[7] :
- $x - 2 = 0 \Rightarrow x = 2$
- $x + 3 = 0 \Rightarrow x = -3$

**Étape 2 : Analyser le signe de chaque facteur**

Pour une fonction affine $ax + b$[7] :
- Si le coefficient directeur $a$ est **positif**, la fonction est croissante : d'abord **négative**, puis **positive**
- Si le coefficient directeur $a$ est **négatif**, la fonction est décroissante : d'abord **positive**, puis **négative**

Pour $(x - 2)$ : coefficient = $+1$ (positif) → ordre des signes : $-$ puis $+$
Pour $(x + 3)$ : coefficient = $+1$ (positif) → ordre des signes : $-$ puis $+$

**Étape 3 : Appliquer la règle des signes**

Le signe d'un produit dépend du nombre de facteurs négatifs[7] :
- Nombre pair de facteurs négatifs → produit **positif** $(+)$
- Nombre impair de facteurs négatifs → produit **négatif** $(-)$

---

## Tableau de signes complet

| $x$ | $-\infty$ | | $-3$ | | $2$ | | $+\infty$ |
|---|---|---|---|---|---|---|---|
| $x + 3$ | | $-$ | $0$ | $+$ | $+$ | $+$ | |
| $x - 2$ | | $-$ | $-$ | $-$ | $0$ | $+$ | |
| $f(x)$ | | $+$ | $0$ | $-$ | $0$ | $+$ | |

**Interprétation :**
- $f(x) > 0$ sur $]-\infty, -3[ \cup ]2, +\infty[$
- $f(x) < 0$ sur $]-3, 2[$
- $f(x) = 0$ pour $x = -3$ et $x = 2$

Niveau: Première
Question: Dresse le tableau de variations de f(x) = x² - 4x + 3 sur R.

Envoi de la requête à Perplexity (Modèle: sonar)...


✅ RÉPONSE DU TUTEUR :

Pour dresser le **tableau de variations** de la fonction \(f(x) = x^2 - 4x + 3\) sur \(\mathbb{R}\), définie au programme de Première, procédons étape par étape. As-tu déjà calculé la dérivée \(f'(x)\) ? Rappelle-toi que les variations sont déterminées par le signe de \(f'(x)\).[1][6]

### Étape 1 : Calcul de la dérivée
La dérivée est \(f'(x) = 2x - 4\).  
Factorisons : \(f'(x) = 2(x - 2)\).[1]

### Étape 2 : Zéros de la dérivée et intervalles
La dérivée s'annule pour \(x = 2\).  
Les intervalles à étudier sont \(]-\infty, 2[\), \(\{2\}\) et \(]2, +\infty[\).[1][6]

### Étape 3 : Signe de \(f'(x)\) et variations
- Pour \(x < 2\), \(f'(x) > 0\) : \(f\) est **strictement croissante**.  
- Pour \(x > 2\), \(f'(x) < 0\) : \(f\) est **strictement décroissante**.  
\(f(2) = -1\) (valeur au sommet, minimum local).[1]

### Tableau de variations complet
```
x       | −∞    |      | 2    |      | +∞
        |       | 2    |      | 2    |
f′(x)   |   +   |  0   |      |   −  |
        | ↗    |      | ↘   |      |
f(x)    |       | −1   |      |
```

Ce tableau montre que \(f\) atteint son minimum en \(x=2\). Peux-tu vérifier le signe de \(f'(x)\) pour \(x=0\) et \(x=3\) ? Trace la parabole pour visualiser ! Bravo pour cette étude classique des quadratiques (coefficient de \(x^2 > 0\)).[1][6]

Niveau: Première
Question: Trace le graphique de la fonction f(x) = -x² + 2x + 3 sur l'intervalle [-2, 4]. Montre-moi une belle courbe lisse.

Envoi de la requête à Perplexity (Modèle: sonar)...


✅ RÉPONSE DU TUTEUR :

Excellente demande ! La fonction **f(x) = -x² + 2x + 3** est une fonction quadratique (parabole ouvrant vers le bas car a = -1 < 0), au programme de **Première** (fonctions affines et quadratiques, Eduscol). Pour tracer son graphique sur l'intervalle **]−2, 4[** de manière lisse, calculons d'abord ses caractéristiques clés.

### Étape 1 : Caractéristiques essentielles
- **Sommet** (point extrémité maximum) : x_S = −b/(2a) = −2/(2×(−1)) = **1**.  
  Puis f(1) = −(1)² + 2(1) + 3 = **4**. Sommet en **S(1 ; 4)** [3][6].
- **Discriminant** Δ = b² − 4ac = 4 − 4(−1)(3) = **16** > 0 (deux racines réelles).  
  Racines : x = [−2 ± √16]/(2×(−1)) → x₁ = −1 et x₂ = 3 [1][2].
- **Image de l'intervalle** : Sur [−2, 4], f(−2) = 3, f(4) = −9. Maximum à 4, minimum à −9.

Qu'as-tu calculé toi-même pour le sommet ou les racines ? Cela t'aide à bien comprendre !

### Étape 2 : Tableau de valeurs (au moins 10 points pour une courbe lisse)
Voici les valeurs calculées précisément (espacées pour fluidité) :

| x   | −2  | −1,5 | −1  | −0,5 | 0   | 0,5 | 1   | 1,5 | 2   | 2,5 | 3   | 3,5 | 4   |
|-----|-----|------|-----|------|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| f(x)| **3**| **3,75** | **4** | **3,75** | **3** | **3,25** | **4** | **3,25** | **3** | **2,25** | **0** | **−1,25** | **−9** |

Remarque la **symétrie** autour de x = 1 (f(1 − t) = f(1 + t)).

### Étape 3 : Graphique lisse
Utilise ces points pour tracer une belle parabole (courbe régulière, symétrique, passant par S(1;4), coupant l'axe des x en −1 et 3).

```
@@@ Parabole f(x) = -x² + 2x + 3 sur ]-2, 4[ | -2,3 | -1.5,3.75 | -1,4 | -0.5,3.75 | 0,3 | 0.5,3.25 | 1,4 | 1.5,3.25 | 2,3 | 2.5,2.25 | 3,0 | 3.5,-1.25 | 4,-9 | domain:-2,4,-10,5 @@@
```

### Étape 4 : Vérification et conseils de tracé
- **Sens de variation** : ↘ sur ]−2, 1[ puis ↗ ? Non : comme a < 0, **croissante** sur ]−2, 1[ (↗), **décroissante** sur ]1, 4[ (↘).
- Pour un tracé main levée : commence par le sommet S, les racines, les valeurs aux bornes, puis relie par une courbe symétrique et régulière (pas de segments droits !).

Essaie de tracer toi-même avec ces points, puis compare au graphique généré. Quelles sont les coordonnées que tu as trouvées en premier ? Bravo pour cette fonction classique, tu progresses bien ! Si besoin, on détaille le tableau de variations complet. 😊
